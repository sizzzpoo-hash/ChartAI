# **App Name**: Chart Alchemist

## Core Features:

- TradingView Chart Integration: Integrate TradingView Lightweight Charts library to display candlestick charts.
- AI Chart Analysis: Use Gemini 2.5 Pro to analyze the candlestick chart data and provide a summary.
- Trade Signal Generation: Generate precise trade signals based on the AI analysis, including entry price range, take profit(s), and stop loss levels using a tool.
- Analysis History: Maintain a history of past AI analysis and trade signals for review.
- Homepage: Main page, to present to the user to key app info.
- Settings Page: Page to allow user customizable setting like chart appearance and risk preference
- Display Candlesticks: Ensure candlesticks are clearly visible on the chart, to comply with the request

## Style Guidelines:

- Primary color: Saturated blue (#3366FF) to reflect trustworthiness.
- Background color: Very light desaturated blue (#F0F5FF) for a clean interface.
- Accent color: Vibrant purple (#A64DFF) for interactive elements and highlights.
- Body and headline font: 'Inter' sans-serif for a modern, machined, objective, neutral look; suitable for headlines or body text
- Code font: 'Source Code Pro' monospace for displaying code snippets in analysis details.
- Use consistent, professional icons for navigation and actions.
- Implement a clean, intuitive layout with clear sections for the chart, analysis, and trade signals.
